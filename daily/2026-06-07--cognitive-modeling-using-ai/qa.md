# Q-A

## Q1: AI 像人就能解释人吗？

**Conclusion:**

不能。行为相似只是入口，机制映射才是解释。

**Formalized relation:**

`像人输出 ≠ 人类机制；模型机制 <-> 人类机制 = 解释入口`

**Reasoning steps:**

- 一个 LLM 答对 false belief 题，只说明它给出了人类会给的答案。
- 这个答案可能来自训练污染、语言统计、任务模板，也可能来自可迁移的心理状态表征。
- 只有说明模型里的什么结构对应人的什么认知过程，行为相似才开始变成科学证据。

**Boundary:**

行为相似不是没价值。它可以作为筛选候选模型的第一步，也可以服务预测任务；但它本身不能证明模型按人的方式在想。

## Q2: 老模型为什么还重要？

**Conclusion:**

老模型刺激贫瘠，但机制干净；新模型刺激丰富，但机制常常没建起来。

**Formalized relation:**

`旧: 窄输入 + 明机制；新: 宽输入 + 黑箱风险`

**Reasoning steps:**

- 早期 naive psychology 概率模型只能处理盒子、篮子、动作位置这类简化变量。
- 它的优点是 belief、goal 等变量明确对应人类心理表征。
- LLM 能读任意社会故事，但通常没有说明内部状态如何对应信念、欲望和目标。

**Boundary:**

旧模型不是更真实。它们的输入过度简化，难以推广到真实世界。作者不是要回到旧模型，而是要把旧模型的机制清楚性带进新 AI。

## Q3: “刺激可计算”改变了什么？

**Conclusion:**

它让模型第一次能直接吃接近人类实验的材料，心理学的比较范围被放大了。

**Formalized relation:**

`真实刺激 -> AI 可处理 -> 大范围人机对照`

**Reasoning steps:**

- 过去模型常要把故事、图像、动作压成几个手工变量。
- 现代 AI 可以处理文字、图像、音频，甚至视频。
- 同一份材料可以给人，也可以给模型，于是研究者能更系统地比较两者反应。

**Boundary:**

刺激可计算不是刺激等同。把视频转成文字、把儿童互动任务转成 prompt，都会改变任务。模型吃得到材料，不等于它经历了人的情境。

## Q4: 映射凭什么成立？

**Conclusion:**

映射靠三件事撑住：训练输入可比、内部机制可指认、评估指标可连接。

**Formalized relation:**

`训练映射 + 机制映射 + 评估映射 -> 认知模型`

**Reasoning steps:**

- 训练映射问：模型学到能力的输入，是否和人类学习输入有可比较关系。
- 机制映射问：模型内部状态、架构或目标，是否对应某个假定的人类认知过程。
- 评估映射问：模型指标怎样对应人类实验里的选择、反应时、判断或神经表征。

**Boundary:**

映射不要求一模一样。好模型可以简化目标系统；但简化必须明说，不能把不匹配藏起来。

## Q5: 训练数据为什么致命？

**Conclusion:**

因为同样的能力，可能由完全不同的学习史长出来。

**Formalized relation:**

`同一行为 = 人类小数据路线 或 AI 超大数据路线`

**Reasoning steps:**

- GPT-3 约用 4000 亿词训练。
- 人类儿童每年语言输入通常少于 1000 万词。
- 如果模型靠多几个数量级的数据才获得能力，它作为儿童学习模型就不能直接类比。

**Boundary:**

数据量不匹配不自动废掉模型。模型仍可用于证明某能力“原则上可学”，也可作为 human simulator；但作为具体人类学习机制，它的证据力会下降。

## Q6: 怎么评估才不自欺？

**Conclusion:**

不要只看 benchmark 分数，要看测量是否有效、对比是否受控、材料是否污染。

**Formalized relation:**

`分数高 -> 能力声明；受控实验 -> 机制推断`

**Reasoning steps:**

- 直接 prompt 模型给判断，可能低估或歪曲模型的潜在知识。
- 模型指标要有 linking hypothesis，说明它如何对应人类反应。
- benchmark 常把长度、难度、领域混在一起，分数差不等于能力差。
- 已发表材料很可能进入训练语料，污染会让“泛化”变成“背题”。

**Boundary:**

受控评估成本高，尤其当模型闭源、不能重训、不能消融时。现实研究可能只能先做弱推断，但弱推断不能包装成强理论。

## Q7: 模拟人和解释人差在哪？

**Conclusion:**

模拟人追求预测准，解释人还要说明为什么准。

**Formalized relation:**

`human simulator = 预测；cognitive model = 预测 + 机制`

**Reasoning steps:**

- AI 可以预测人类道德判断、实验效果大小或 trial-by-trial 选择。
- 这对材料预试、实验筛选和社会科学研究很有用。
- 但如果不知道模型为什么预测准，它只是强预测器，不是心理理论。
- Centaur 预测表现强，但底座吃过约 15 万亿 token，这让它很难直接代表人类学习史。

**Boundary:**

预测不是低级活。好的 human simulator 可以成为认知模型的前一步；只是不能把第一步当终点。

## 收口

这篇文章真正砸下来的钉子是：AI 对心理学的价值不在于它“像人”，而在于它能不能被改造成可映射、可干预、可复现的心智模型。
